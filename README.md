### Currently building

🔧 **[tiko-di](https://github.com/tomas-samek/tiko-di)** —
Compile-time dependency injection framework for Java 17+.
All wiring validated and generated by an annotation processor;
no reflection, no classpath scanning, no runtime DI surprises.
Ships with a built-in event bus that swaps transparently
between local and distributed implementations.

🧠 **[trie-memory](https://github.com/tomas-samek/trie-memory)** —
Experimental MCP memory server in Rust. Append-only hierarchical
memory with explicit epistemic states (Answer / Partial / Unknown)
— designed not to fabricate when it doesn't actually remember.
Research PoC built on a delta-encoded recognition trie paired with
a concept store.
